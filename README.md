## Getting Started  

### Run Locally  

Clone the project

```bash
  git clone https://github.com/dimitrisAvgoustopoulos/react-project-management-app.git
```

Go to the project directory

```bash
  cd react-project-management-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
